# Reconciliation Transaction — Frontend Business Logic

บันทึกการตัดสินใจและขอบเขตงานฝั่ง frontend

## In-Scope

- 3-panel layout: Preparation, Preparation + Machine Data, Header Card from Machine
- Scan Header Card barcode
- Filter by Header Card and denomination
- Edit/Delete ReconcileTran
- Reconcile detail popup with denomination breakdown
- OTP verification flow (Supervisor + Manager)
- Cancel Reconcile with report
- 4 CSS variants (Unfit, Unsort CC, CA Member, CA Non-Member)
- Reconcile count summary
