# Reconciliation Transaction — Popup Documentation

## Popup List

### 1. แก้ไข Header Card (Edit Header Card)
- Fields: Header Card Code, Remark
- Buttons: ยกเลิก (Cancel), บันทึก (Save)

### 2. กระทบยอด (Reconcile Detail)
- Denomination breakdown table
- Tabs: ชนิดราคา (denomination types)
- Compare: Preparation qty vs Machine qty
- Buttons: ยกเลิก, กระทบยอด (Reconcile)

### 3. ตรวจสอบยอด (Reconcile Verification)
- Header card info display
- Denomination comparison grid
- Warning indicators for mismatches

### 4. OTP Supervisor Verification
- Select Supervisor dropdown
- OTP input field
- Send OTP button
- Verify button

### 5. Confirm Success
- Success icon
- ดำเนินการเรียบร้อย message
- ปิด (Close) button

### 6. ยกเลิกกระทบยอด (Cancel Reconcile)
- Requires Manager OTP
- Select Manager dropdown
- OTP verification

### 7. Cancel Reconcile Report
- Print format for cancel reconciliation
- Report details and denomination table

### 8. Warning Alert (Manual Key-in)
- Alert for Quality/Output mismatch
- Option to proceed with manual key-in
