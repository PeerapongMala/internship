# Reconciliation Transaction — Backend Business Logic

บันทึกการตัดสินใจและขอบเขตงานฝั่ง backend

## In-Scope

- API สำหรับกระทบยอดธนบัตร (Reconciliation Transaction)
- Scan Header Card → สร้าง ReconcileTran
- CRUD สำหรับ ReconcileTran (Edit, Delete)
- Reconcile action: เปรียบเทียบ Preparation vs Machine data
- Cancel Reconcile
- Summary count
- Status management: Reconciliation(11) → Reconciled(13) → Approved(16) → Verify(17)
