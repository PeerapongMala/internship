# FAQ Feature - Requirements & Flow

## สรุป Requirements

### ส่วนที่ 1: FAQ Management Page (`/faq`)

| Requirement                 | Status | หมายเหตุ                         |
| --------------------------- | ------ | -------------------------------- |
| Head Admin แก้ไข FAQ เองได้ | ✅     | เพิ่ม/แก้ไข/ลบ FAQ               |
| แบ่งหมวดหมู่ FAQ            | ✅     | ราคา, บริการ, นัดหมาย, โปรโมชั่น |
| ค้นหา FAQ                   | ✅     | Search box                       |
| AI ช่วยกรอกคำตอบ            | ✅     | ปุ่ม "AI ช่วยกรอก" (simulate)    |

### ส่วนที่ 2: Chat Question Logs (Tab คำถามจากแชท)

| Requirement              | Status | หมายเหตุ                             |
| ------------------------ | ------ | ------------------------------------ |
| ดึง log คำถามจากแชท      | ✅     | Mock data (จะเชื่อม DB ภายหลัง)      |
| แสดงรายการคำถามแต่ละวัน  | ✅     | มี timestamp                         |
| Checkbox เลือกคำถาม      | ✅     | Single & multi select                |
| Bulk actions             | ✅     | เลือกทั้งหมด, ลบหลายรายการ           |
| ปุ่ม "เพิ่มเป็น FAQ"     | ✅     | เปิด modal กรอกคำตอบ                 |
| Text input พิมพ์คำตอบ    | ✅     | ใน modal                             |
| AI ช่วยกรอก + ปุ่มยืนยัน | ✅     | ปุ่ม "AI ช่วยกรอก" + "เพิ่มเป็น FAQ" |

### ส่วนที่ 3: Settings (Tab ตั้งค่า)

| Requirement             | Status | หมายเหตุ                  |
| ----------------------- | ------ | ------------------------- |
| จัดการหมวดหมู่          | ✅     | เพิ่ม/ลบ category         |
| ข้อความ intro สำหรับแชท | ✅     | แก้ไขได้                  |
| Preview FAQ ในแชท       | ✅     | แสดงตัวอย่าง + auto-reply |

## สิ่งที่ Implement แล้ว vs ยังไม่ได้ทำ

### ✅ Implement แล้ว (Frontend Mock)

- หน้า FAQ Management พร้อม 3 tabs
- CRUD FAQ (เพิ่ม/แก้ไข/ลบ)
- Chat Question Logs พร้อม checkbox + bulk actions
- AI ช่วยกรอก (simulate)
- Preview FAQ ในแชท + auto-reply
- จัดการหมวดหมู่

### ⏳ ยังไม่ได้ทำ (ต้องเชื่อม Backend)

- เชื่อม Database จริง
- ดึง log คำถามจาก chat system
- บันทึก FAQ ลง Database
- เชื่อม AI จริงสำหรับช่วยกรอก
- ส่ง FAQ links ไปยัง chat จริง
