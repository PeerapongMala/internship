ตอนนี้จะเริ่มโปรเจคใหม่ เกี่ยวกับ เว็ปบรรเทาความกลัวเกี่ยวกับสัตว์ มีด่าน มีเหรียญ reward admin user doctor และผู้ป่วย

user มี 2 แบบ
1 จาก code ของหมอ
2 จาก ผู้ใช้ทั่วไปที่สมัครเข้ามาใช้เว็ป

doctor จะเก็บบันทึกจากผู้ป่วยได้
และความคืบหน้าของด่านที่ผู้ป่วยเล่นจะถูกเก็บ ใน doctor ด้วย

ผู้ใช้ทั่วไป สามารถlogin และ signupโดย username password และเก็บfull-Name email
ถ้าหากเป็นผู้ป่วย สามารถเข้าผ่านcode ที่ doctor generate มาให้
ผู้ป่วยกรอบบันทึกของด่านได้ และ จะถูกเก็บไว้ในประวัติ และdoctor ก็ได้บันทึกจากผู้ป่วยด้วย

ด่านจะแบ่งสัตว์เป็นหลายหมวดหมู่ แต่ละหมวดก็จะมีสัตว์หลายชนิด เช่นหมวดสัตว์บก สัตว์ปีก สัตว์เลี้ยงลูก
ด่านแต่ละด่านจะดึงข้อมูลรูปภาพ จาก db มาใช้
ความคืบหน้าของด่านจะถูกเก็บข้อมูลไว้ตาม user
ถ้าเป็นของผู้ป่วยทั่วไปจะถูกเก็บไว้ใน ประวัติ user ด้วยว่าเล่นผ่านมั้ยได้
